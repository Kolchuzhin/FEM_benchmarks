## conductor with Boundary Layer Mesh in ANSA
conductor 8x8 mmxmm
Basic Parameters of layers:
* First height: 0.20
* Growth factor: 1.20
* Number of layers: 5
* Additional outer layers: 0
* Last aspect: 0.80
